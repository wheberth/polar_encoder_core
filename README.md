# Simple combinational Polar Encoder

The main putpose of this repo is to serve as placeholder for useful scripts for FPGA development.
In this repo you will find:
* Directory structure for simple HDL projects;
* Makefile scripts for simulations of HDL using Xivado tools;
* Scripts to run Vivado simulation inside docker/podman container;

## Requirements
* Vivado (or in a container)
* GNU make
